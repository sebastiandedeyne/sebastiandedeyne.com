Putting this in a blog post because I _always_ forget.

To view PHP logs from Laravel Valet:

{.short}
```sh
open ~/.config/valet/Log/php-fpm.log
```

To view NGINX logs from Laravel Valet:

{.short}
```sh
open ~/.config/valet/Log/nginx-error.log
```
